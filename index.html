scrollHash.js

A Applications Sample of location.hash.

How To Use?

Bind / Add classname "scrollHash" to your element target.

Html:

<div class="scrollHash">...</div>

jQuery / javascript:

<script>
$(function(){
    wsc.scrollHash();
});
</script>

Options:

scrollHash.js : (Options)
Options Name Default Value Type Comment
scrollHash true (Boolean) When scroll to SHclassName element, change window.Hash.
clickHash true (Boolean) When click SHclassName element, change window.Hash.
SHclassName "scrollHash" (Boolean) ScrollHash Target.
range 50 (Number) When window scroll Between element's top +Rage -Rage, change window.Hash.
